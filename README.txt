Instal Grunt:
Open Node Terminal => cd Within this folder => npm install grunt --save-dev
Node terminal => cd node_modules => npm install grunt-contrib-jshint --save-dev
                                    npm install grunt-contrib-watch --save-dev
                                    npm install grunt-contrib-connect --save-dev

Right click DragDropBuilder folder => GitBash Here => Git terminal type: grunt server
A browser window will open for localhost:9000

To change the guitar body image edit the html file -> line 136 -> Change the last number (stratBody*[]*)
